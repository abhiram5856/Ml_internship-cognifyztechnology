## ⭐ Task 1: Predict Restaurant Ratings

### Objective

Build a machine learning model capable of predicting a restaurant's aggregate rating.

### Workflow

* Data Cleaning and Preprocessing
* Feature Selection
* Train-Test Split
* Regression Model Development
* Performance Evaluation

### Techniques Used

* Pandas
* NumPy
* Scikit-learn
* Regression Algorithms

### Evaluation Metrics

* Mean Squared Error (MSE)
* Mean Absolute Error (MAE)
* R² Score
